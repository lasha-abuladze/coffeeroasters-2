# ☕ My First Big Project: Coffeeroasters Website

Hi! This is my first full multi-page website. I built this to practice my skills in HTML, SCSS, and JavaScript. It’s a coffee subscription site where you can actually "Create your plan" by clicking through a quiz.

[🔗 Click here to see the live website!](https://lasha-abuladze.github.io/coffeeroasters-2/)

## 📝 What I learned
This project was a huge challenge for me. Here are the main things I practiced:
- **Responsive Design:** Making sure the website looks good on my phone, my tablet, and my laptop.
- **SCSS:** I learned how to use variables and keep my CSS organized in different files.
- **JavaScript:** I wrote the logic for the "Create your plan" page so it reacts when you click the options.
- **Git & Commits:** I used "Conventional Commits" (like `feat:`, `style:`, and `refactor:`) to keep my history clean.

## 🛠️ Tools I used
- **HTML5** (for the structure)
- **SCSS/CSS** (for the styling)
- **Vanilla JavaScript** (for the interactive parts)
- **VS Code** (my code editor)


## 🚀 How to see it locally
1. Clone this folder.
2. Open `index.html` with Live Server in VS Code.
3. Check out the "Create Your Plan" page!
